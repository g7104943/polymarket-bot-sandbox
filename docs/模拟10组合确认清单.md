## 三、本机自检：是否只有这 10 个组合在跑

在项目根目录执行下面命令做快速检查。

### 1. 看旧模型 5 个：应有 5 个 Python 写预测 + 5 个交易进程

```bash
# 应有 5 个 prediction_writer 进程（A/B/C/E/F，不要有 D）
ps aux | grep -E "prediction_writer.*predictions_[ABCEF]\.json" | grep -v grep

# 应有 5 个交易进程（predict-trade:A/B/C/E/F，不要有 D）
ps aux | grep -E "predict-trade:(A|B|C|E|F)" | grep -v grep
```

### 2. 看 GRU 5 个：应有 5 个 GRU 写预测 + 5 个交易进程（ETH 两个组合共用一个 Python）

```bash
# 应有 4 个 prediction_writer_gru（eth/xrp/btc/sol 各一；eth_54 与 eth_55 共用 eth 的 Python）
ps aux | grep prediction_writer_gru | grep -v grep

# 应有 5 个 GRU 交易进程
ps aux | grep -E "predict-trade:gru_(eth_55|xrp_55|btc_55|eth_54|sol_52)" | grep -v grep
```

### 3. 确认没有 D（logs_eth_10_80）

```bash
# 应无输出；若有输出说明还有 D 在跑，需执行 ./停止三个版本.sh 再查
ps aux | grep -E "predictions_D|predict-trade:D" | grep -v grep
```
